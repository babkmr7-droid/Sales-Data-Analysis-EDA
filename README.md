# 📊 Sales Data Analysis & Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project focuses on Data Cleaning, Exploratory Data Analysis (EDA), and Business Insight Generation using Python and various data analysis libraries.

The primary objective of this project is to analyze customer purchasing behavior, sales trends, product performance, and demographic patterns to generate meaningful business insights.

## 🚀 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

The dataset contains customer purchasing information including:

- Marital Status

# 🔍 Key Business Insights

### Customer Insights

- Certain age groups contribute significantly to total sales.
- Customer spending behavior varies across demographics.
- Purchase patterns differ based on gender and marital status.

### Sales Insights

- Top-performing states generate the highest revenue.
- Some product categories outperform others in both sales and order volume.
- A small group of customers contributes a significant portion of total sales.

### Business Recommendations

- Focus marketing campaigns on high-performing customer segments.
- Improve promotions for underperforming product categories.
- Utilize state-wise sales analysis for targeted business expansion.
